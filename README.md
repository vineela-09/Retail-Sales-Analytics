# 🛒 Retail Sales & Inventory Analytics (SQL)

## 📌 Project Overview

Retail Sales & Inventory Analytics is a SQL-based data analysis project developed using the Superstore Sales Dataset. The project demonstrates SQL skills by analyzing retail sales, customer behavior, product performance, and regional trends using MySQL.

The objective of this project is to solve real-world business problems through SQL queries, views, stored procedures, triggers, and indexing techniques.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify top-performing products and categories.
- Analyze customer purchasing behavior.
- Evaluate regional sales and profitability.
- Generate business insights using SQL.
- Optimize query performance using indexes.

## 🚀 How to Run the Project

1. Clone this repository.

```
git clone https://github.com/yourusername/Retail-Sales-Analytics.git
```

2. Open MySQL Workbench.

3. Execute SQL files in the following order:

```
01_create_database.sql

02_create_tables.sql

03_insert_data.sql

04_business_queries.sql

05_views.sql

06_stored_procedures.sql

07_triggers.sql

08_indexes.sql
```

4. Import the Superstore CSV dataset into MySQL before running the analysis queries.
